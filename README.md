# Full Stack MERN Project

---

React + Redux, Node, Express, MongoDB

---

## Installation

```
$ git clone https://github.com/PavelKrokhmal/memories-project
$ cd ./memories-project
$ npm init
```

## Running

Setup .env variables

Run server using:

```
$ cd server
$ npm run start
```


Run client using:

```
$ cd client
$ npm run start
```
